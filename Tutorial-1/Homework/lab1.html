Course Summary

ICT726 introduces the fundamentals of Web Development. Students learn HTML5, semantic elements, hyperlinks, images, and webpage structure.

Course Expectations

  • Attend lectures and tutorials.
  • Complete practical exercises.
  • Submit assignments on time.

Textbook

The recommended textbook covers HTML5, semantic markup, hyperlinks, images, and basic web development concepts.

Introduction to the Internet

The Internet is a worldwide network that allows computers to communicate and exchange information.

Client-Server Model

In the client-server model, the client sends a request and the server processes the request and returns a response.

HTML Tags

HTML tags define the structure of a webpage using headings, paragraphs, images, links and other elements.
